##### HTTP Stands for Hyper Text Transfer Protocol which is a web protocol that views and shares web content for user.

http is considered as a non secure protocol due to its capablity of not encrypting serfed data which makes it less secure compared to HTTPS.

#### HTTPS stands for Hyper Text Transfer Protocol Secure, this is a web protocol just like http but it encrypts the data which makes it much more secure and reliable.
